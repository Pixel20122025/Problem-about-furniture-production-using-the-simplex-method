Задача про виробництво меблів

Умова задачі

Фабрика виробляє столи і стільці:

  • Прибуток: стіл = 120 грн, стілець = 40 грн
  • Деревина на складі: 350 кг
  • Деревина на 1 стіл: 15 кг, на 1 стілець: 4 кг

Математична модель:

x₁ = кількість столів
x₂ = кількість стільців
Максимізувати Z = 120x₁ + 40x₂
Обмеження:
15x₁ + 4x₂ ≤ 350
x₁, x₂ ≥ 0

Розв'язок симплекс-методом

Перетворюємо нерівність у рівність, додаючи змінну запасу s:

15x₁ + 4x₂ + s = 350, s ≥ 0

Цільова функція:

Z - 120x₁ - 40x₂ = 0

Початкова симплекс-таблиця

Базис x₁ x₂ s RHS
s 15 4 1 350
Z -120 -40 0 0

Вибираємо x₁ як вхідну змінну (найбільший негативний коефіцієнт). Вихідна змінна: s (350/15 ≈ 23.33)

Після першого кроку:

x₁ = 23.33, x₂ = 0, s = 0
Z = 120*23.33 + 40*0 ≈ 2800

Перевірка обмеження: 15*23.33 + 4*0 = 350

Отже, оптимальне рішення без цілочисельності:

x₁ = 23.33, x₂ = 0, Z ≈ 2800
